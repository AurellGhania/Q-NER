## QR Scanner App

A Mobile-based QR Scanner and Generator App.



## Features

- QR Scanner Page
- QR Generator Page


## Tech Stack

**Front-End:** 

- Dart v.3.2.4 (stable)
- Flutter v.3.24.5 (stable channel)

**Dependencies:** 

- mobile_scanner
- pretty_qr_code


## Usage

Needs user's permission to allow their camera.
